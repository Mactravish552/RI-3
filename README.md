# Exercício RI3

Repositório com a resolução do exercício de Técnicas de Programação RI3, utilizando JavaScript.

*P.S.: No arquivo do RI existe um easter egg.*

## 📋 Pré-requisitos

Para rodar o projeto, você precisará do Node.js instalado na sua máquina.

### 1. Instalar o Node.js (Windows)
```bash
winget install OpenJS.NodeJS.LTS
```

### 2. Como inicializar os projetos
Para executar diretamente pelo terminal, use o comando node seguido do nome do arquivo:
```Bash
node nomeDoArquivo.js
```
(Exemplo: node RI3.js)

### 4. Descrição dos Exercícios
RI3: Evolução do Cadastro

Evolução do RI2, adicionando a Empresa que pode ter múltiplos Clientes e Telefones, mas apenas um Endereço associado a ela.
